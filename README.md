以MCMC進行簡單線性回歸
The Bayesian two-feature regression confirms that alcohol has a positive effect on wine quality, while volatile acidity has a negative effect. Both effects are supported by posterior intervals that do not include zero.

Although the Bayesian model shows slightly lower predictive performance compared to the frequentist baseline, it provides well-calibrated uncertainty estimates, as demonstrated by a 95% predictive interval coverage of 94.3%.

Posterior predictive analysis reveals that the model captures the overall distribution of wine quality but smooths out its discrete nature, reflecting the Gaussian likelihood assumption.

Overall, this project demonstrates that while frequentist models may offer slightly better point predictions, Bayesian methods provide richer insights by quantifying uncertainty in both parameters and predictions.
